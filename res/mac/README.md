`AppIcon.icns` & `Assets.car` is generated from `Apple Icon.icon` by making a dummy Xcode project, importing `Apple Icon.icon`, then checking "Include All App Icon Assets" under "General -> App Icons and Launch Screen" and "Build Settings -> Asset Catalog Compiler" the icon name should also be set under those options to whatever the file name is. Finally just build the app and grab the 2 files from the built Hello World .app.
